# Generated text and code must be in English

All generated text must be written in English, including
code. This covers source code (identifiers, comments,
docstrings, log/error messages), documentation, README
content, commit messages, and any prose produced for the
project.

**Why:** the workshop and reference application target an
international audience; a single language keeps the codebase
and materials consistent and shareable.

**How to apply:** always produce English output regardless
of the conversation language. When the user writes in another
language (e.g. French), still generate all artifacts in
English.
